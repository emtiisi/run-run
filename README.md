# Neon Runner

A 3D Infinite Runner game built with Three.js and Vite.

## How to Play
1. Install dependencies: `npm install`
2. Start the game: `npm run dev`
3. Open the link (usually http://localhost:5173) in your browser.

## Controls
- **Left / A**: Move Left
- **Right / D**: Move Right
- **Space**: Jump / Start / Restart

## Features
- Infinite 3D world
- Increasing speed
- Obstacles (Red Cubes)
- Power-ups:
    - **Blue Sphere**: Shield (One-time invincibility)
    - **Yellow Cone**: Speed Boost (Score multiplier)
